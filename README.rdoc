== Clutch

This is Clutch, a competitive gaming website for gamers to create their clans and challenge other clans in various ladders across different games to matches.
The languages and frameworks used are :

- Rails API
- Ruby
- Javascript
- AngularJS
- Various Javascript libraries e.g. Underscore, JQuery.
- HTML5 and CSS/Sass
- Bootstrap
- Postgres
- Deployment on Heroku
